# Магазин (пет-проект)
Использует:
1. fastApi
2. sqlaclhemy
3. pydantic

## Модели
Модели позволяют сформировать базу данных под работу с объектами категорий/продуктов

## Схемы
Pydantic схемы занимаются валидацией входящих/исходящих данных в API

## Репозитории
Репозитории содержат логику прямой работы с БД, например - запросы на получение товаров

## Сервисы
Сервисы позволяют правильно обработать запрос, а после вернуть валидный ответ
